## How To Use

The website is published [here](https://pras306.github.io/gpt3-modern/)

Clone the project 

```bash
  git clone https://github.com/pras306/gpt3-modern.git
```

Go to the project directory

```bash
  cd gpt3-modern
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
